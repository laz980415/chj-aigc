# Frontend Admin

This is the separated Vue 3 admin console for `chj-aigc`.

## Requirements

- Node.js `20.19.0`

## Development

```powershell
npm install
npm run dev
```

The dev server runs on `http://127.0.0.1:5173` and proxies `/api` to the Spring Boot backend on `http://127.0.0.1:8080`.

## Build

```powershell
npm run build
```
